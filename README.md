## Virtual environment
### 가상 환경 설치부터 모듈 설치까지 한번에 하는 방법
#### 명령어
> conda env create -f environment.yml

#### environment.yml

> name: {environment name}
> channels:                             # 이건 아직 잘 모르겠음
> dependencies                          # conda 패키기 설치 시작
>   - {conda package name}={version}
>   - pip:                              # pip 패키지 설치 시작
>     - {pip package name}={version}
